# إصلاح نصوص "أسباب الانتشار" المخفية ✅

## المشكلة:
**"في خانات أسباب الانتشار الكلام غير ظاهر الصفحة سوداء والكلام مختفي"**

- النصوص داخل أقسام `.pest-body` (أسباب الانتشار) غير مرئية
- الكلام مختفي أو صعب القراءة  
- قد تكون الخلفية سوداء أو النصوص بدون ألوان

## السبب:
عناصر `h2` و `p` و `li` داخل `.pest-body` لم تملك تنسيق خاص لحدد الألوان والأوزان.

## الإصلاح المنفذ:

### 1. إضافة CSS جديد في `articles.css`:
```css
/* Pest body text styling */
.pest-body h2 {
  color: #0d5d0d;           /* أخضر الشركة */
  font-weight: 700;         /* جريء */
  margin-top: 20px;
  margin-bottom: 12px;
  font-size: 1.4rem;
}

.pest-body p {
  color: #1a1a1a;           /* أسود داكن */
  line-height: 1.8;
  margin-bottom: 15px;
  font-weight: 500;         /* متوسط - قراءة أفضل */
}

.pest-body ol, .pest-body ul {
  color: #1a1a1a;
  margin-left: 20px;
  margin-bottom: 15px;
}

.pest-body li {
  color: #1a1a1a;
  margin-bottom: 10px;
  line-height: 1.8;
  font-weight: 500;
}

.pest-body strong {
  color: #0d5d0d;           /* أخضر للكلمات المهمة */
  font-weight: 700;
}
```

### 2. الملفات المحدثة:
✅ `articles.css` - إضافة 5 selectors جديد
✅ `archive/articles.css` - نفس الإضافة
✅ `index.single.html` - إضافة CSS مدمج

## التفاصيل الفنية:

| العنصر | اللون | الوزن | الحجم |
|------|------|------|------|
| `.pest-body h2` | #0d5d0d (أخضر) | 700 (جريء) | 1.4rem |
| `.pest-body p` | #1a1a1a (أسود) | 500 | 1rem |
| `.pest-body li` | #1a1a1a (أسود) | 500 | 1rem |
| `.pest-body strong` | #0d5d0d (أخضر) | 700 | 1rem |

## النتيجة:

✅ **h2 (العناوين):** أخضر داكن جريء → مرئية جداً
✅ **p (النصوص):** أسود داكن متوسط → قراءة سهلة
✅ **li (العناصر):** أسود داكن متوسط → واضحة
✅ **strong (الكلمات المهمة):** أخضر جريء → تميز

## الصفحات المتأثرة:

1. `articles.html` - قسم الصراصير (بق الفراش، القوارض، إلخ)
2. `articles/cockroaches.html` - صفحة الصراصير
3. `articles/bedbugs.html` - صفحة بق الفراش
4. صفحات الآفات الأخرى (جميع ملفات articles/*.html)

## اختبار الإصلاح:

```bash
# 1. شغل الخادم
npm run start

# 2. افتح أي صفحة آفة (مثل articles.html)
# 3. انقر على "اقرأ المزيد" لتوسيع القسم
# 4. تحقق من:
   - ✅ h2 أخضر ومرئي
   - ✅ النصوص سوداء وقابلة للقراءة
   - ✅ القائمة المرقمة واضحة
   - ✅ الكلمات المهمة خضراء وجريئة
```
